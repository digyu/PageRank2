Readme for Page Rank Program

1. Import project into Jave IDE, preferably Eclipse.
	a) In Eclipse, File -> Import -> Existing Project into Workspace
	   and select PageRank project.
2. Open PageRank.java
3. If la4j not present, download it on  http://la4j.org/
4. To add to eclipse, Right click on PageRank folder in the explorer ->
   click build path -> Configure build path -> add eternal libraries
5. Choose the la4j jar file from download location, click open and then okay.
6. Run PageRank.java and results are in console.
